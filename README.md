windows

1. Descarga e instala Python desde:

   * [https://www.python.org/downloads/windows/](https://www.python.org/downloads/windows/)
   * **Marca la casilla "Add Python to PATH"** durante la instalación.

2. Abre **Símbolo del sistema (CMD)** y comprueba que funciona:

```cmd
python --version
```

Si no funciona, prueba:

```cmd
py --version
```

3. Instala Pygame:

```cmd
pip install pygame
```

O si `pip` no funciona:

```cmd
py -m pip install pygame
```

4. Guarda tu archivo `El_farol_rojo.py` en una carpeta y ejecútalo desde esa carpeta:

```cmd
python El_farol_rojo.py
```

O:

```cmd
py El_farol_rojo.py
```
